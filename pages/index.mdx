# __Introducing Riffy__
**Riffy** is a pro lavalink client. It is designed to be simple and easy to use, with a focus on stability and more features.

## __Features__
- Supports lavalink v3 and v4 (active)
- Supports all popular music platforms, including youtube, soundcloud, to use music platforms like spotify and more... make sure you have [Lavasrc](https://github.com/topi314/LavaSrc) plugin in your lavalink server.
- Autoplay support for youtube, soundcloud and spotify.
- Supports discord.js, eris & more.. (adjust as needed)
